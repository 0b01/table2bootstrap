table2bootstrap

Maintained by Ricky Han. rhan@nxtstp.com This tool converts old school table based layout to bootstrap responsive layout.

Steps:

  1. log in
  2. use "Organize Parts" tool to allow dragging and dropping components.
  3. rearrange if necessary ( see Creating n2:DroppableZone )
  4. most Generic Textboxes are table-based and the primary target of conversion.
  5. click edit part; html on the upper right corner; determine if the code is <table>; if so, copy and paste to the converter.
  6. If the edit part doesnt show up. The component is most likely a custom part in the Parts folder (see Editing parts)
  7. copy the converted markup back to the dialog box.

Post-conversion:

each column of the table has two classes to determine the column width. One for mobile (xs) and one for desktop (sm). Column numbers add up to 12. If you want the column to be full width on mobile, change xs to 12. For desktops, adjust as needed.

For building complex interfaces, this tool will serve as a starting point and make your job significantly easier.

Creating n2:DroppableZone

These are anchors for dragging and dropping components. Change the layout files (/Templates/UI/Layouts/) and add corresponding definitions in designer.cs files.

Editing Parts

Some parts are table based. Convert those accordingly. Then change the definitions based on id in designer file from HtmlTable to HtmlGenericControl(beware of camel case)

Setting up tool

Chances are, if you are reading this, you've already set it up. This is a simple self-contained flask application that can be deployed on any server, local or remote. Just run sudo ./run.py and you are all set. Dependencies are flask, cssutils and bs4.
